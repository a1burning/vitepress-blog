# vitepress-blog
使用VitePress重构blog项目

## 参考网站
[VitePress 官方网站](https://vitepress.dev/zh/)

## 启动本地项目

```bash
pnpm run dev

pnpm vitepress dev docs
```

## 构建

```bash
pnpm run build
```

## 预览
```bash
pnpm run preview
```

## 发布
```bash
pnpm run deploy-csf
```

或者build之后直接

```bash
pnpm run deploy-rsync
```
